## 👋 Sobre mí

Network & Systems Administrator en Grupo Hoteles Playa, certificado MikroTik **MTCNA**
y **MTCRE** (ASIR).

Diseño, despliego y automatizo infraestructura de red y sistemas para entornos reales,
multi-sede: segmentación de red y firewall perimetral, autenticación centralizada
(RADIUS/LDAP y Active Directory), monitorización y observabilidad, backup y
recuperación ante desastres, y automatización de despliegues con Ansible.
